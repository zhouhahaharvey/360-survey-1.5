## 1. 文档概述

### 1.1 产品背景
360度人才调研系统是一款面向企业HRBP和员工的人才评估工具，支持晋升评估、转正评估、新Leader考察、绩效考核等多种场景。

### 1.2 目标用户
- **HRBP/管理员**: 发起调研、管理进度、查看报告
- **评估人（员工）**: 填写评估、查看历史、个人成就
- **被评估人**: 查看自己的评估报告

### 1.3 版本说明
v1.5版本主要新增双视角调研列表、高级筛选功能，将报告中心整合到调研详情页，优化员工填写界面底部按钮布局，支持草稿保存和自动进入下一份评估。

---

## 2. 核心流程

### 2.1 HRBP发起流程
```
选择场景 → 导入人员 → 确认关系 → 提交审核 → 管理员审核 → 发起调研
   Step1      Step2       Step3        审核中      审核通过      收集中
```

### 2.2 管理员审核流程
```
待审核列表 → 批量审核 → 选择推送方式 → 发起调研
              ↓
         立即推送 / 加入待推送池 → 合并推送
```

### 2.3 员工填写流程
```
企微消息 → 待办列表 → 选择评估 → 填写问卷 → 提交 → 进入下一份
```

## 5. 技术规格

### 5.1 技术栈
- **UI框架**: Bootstrap 5.3
- **图标库**: Bootstrap Icons 1.11
- **图表**: Chart.js (仪表盘)
- **数据存储**: LocalStorage + DataStore 封装
- **导出**: html2canvas + jsPDF (预留)

### 5.2 浏览器兼容
- Chrome / Edge / Safari / Firefox 最新版本
- iPhone Safari (iOS 14+)
- Android Chrome

### 5.3 数据规范

#### ID格式统一
```
survey_{status}_{index}
- survey_completed_0 ~ survey_completed_3
- survey_active_0 ~ survey_active_4
- survey_pending_0 ~ survey_pending_11
- survey_pushpool_0 ~ survey_pushpool_11
- survey_draft_0 ~ survey_draft_2
```

#### 状态流转
```
draft → pending_review → relation_approved → active → completed
              ↓
         rejected → draft
```
