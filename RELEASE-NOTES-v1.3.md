# 360度人才调研系统 v1.3 发布说明

**发布日期**: 2026-03-25  
**版本号**: v1.3  
**上一版本**: v1.2

---

## 🎯 版本概述

v1.3版本主要聚焦于**员工端体验优化**，简化导航结构，重构个人中心，完善批量操作功能。

---

## ✨ 新增功能

### 1. 员工端重构
- **底部导航简化**: 从4项减少为3项（待办/历史/我的）
- **独立历史页面**: 新增 `history.html`，按月份分类展示已完成调研
- **个人中心重构**: 合并成就中心，包含徽章、证书、组织贡献

### 2. 批量操作完善
- **批量终止调研**: 管理员可勾选多个调研批量终止
- **批量催办**: 催办抽屉支持批量选择后一键催办
- **批量导出**: 报告中心支持批量下载

### 3. 筛选功能增强
- **多维度筛选**: 状态、场景、部门、时间
- **搜索功能**: 支持调研名称/ID/负责人搜索
- **重置筛选**: 一键清空所有筛选条件

## 📱 员工端导航

```
┌─────────────┬─────────────┬─────────────┐
│    📋待办    │    ⏰历史    │    👤我的    │
│  (调研列表)  │  (已完成)   │  (成就中心)  │
└─────────────┴─────────────┴─────────────┘
```

**说明**:
- **待办**: 待完成的调研，支持场景筛选
- **历史**: 已完成的调研，按月份分类
- **我的**: 个人信息、徽章、证书、组织贡献

## 🔐 管理员功能

访问调研列表时添加参数启用:
```
admin/survey-list.html?admin=true
```

**功能**:
- 查看所有调研状态
- 批量终止调研
- 查看操作日志
- 系统级配置
