# 360度人才调研系统 v1.5 发布说明

---

**版本号**: v1.5  
**发布日期**: 2026-04-10  
**上一版本**: v1.4

---

## 🎯 版本概述

v1.5版本聚焦于**调研列表体验升级**和**界面细节优化**，主要新增双视角切换查看、高级筛选功能，将报告中心整合到调研详情页，优化员工填写界面底部按钮布局。

### 核心变更
- 📊 **双视角调研列表** - 支持按被评估人或评估人视角切换查看
- 🔍 **高级筛选** - 状态、场景、截止日期多维度筛选
- 📄 **报告整合** - 报告功能合并到调研详情页，支持导出
- 📝 **员工界面优化** - 底部按钮布局调整，支持草稿保存
- 🗂️ **菜单优化** - 待推送池位置调整，移除报告中心独立入口

---

## ✨ 新增功能

### 1. 双视角调研列表
**文件**: `admin/survey-list.html`

- 新增视角切换按钮（被评估人视角 / 评估人视角）
- 被评估人视角：按被评估人聚合展示调研数据
- 评估人视角：按评估人聚合展示其需要评估的同事列表
- 搜索框根据当前视角自动切换搜索类型

### 2. 高级筛选功能
**文件**: `admin/survey-list.html`

新增筛选维度：
- **状态筛选**: 全部 / 待审核 / 收集中 / 已完成 / 草稿
- **场景筛选**: 全部 / 晋升评估 / 转正评估 / 新Leader / 绩效考核
- **截止日期**: 全部 / 未来3天内 / 未来7天内 / 未来30天内 / 已过期
- **搜索**: 根据当前视角搜索被评估人或评估人姓名/工号

### 3. 报告预览整合
**文件**: `admin/survey-detail.html`, `admin/report-editor.html`

- 移除独立的报告中心页面 (`admin/reports.html`)
- 调研详情页新增报告预览卡片
- 支持导出 PDF / Excel / Word 格式
- 报告编辑器支持从调研数据动态渲染内容

### 4. 员工填写界面优化
**文件**: `respondent/survey-form.html`

底部按钮重新布局：
- **无法填写**（左）- 超小灰色文字，避免误触
- **保存**（中）- 次要按钮，支持保存草稿
- **提交**（右）- 主要按钮，提交后自动进入下一份

新增功能：
- 草稿自动保存和恢复
- 提交后自动进入下一份待办评估

### 5. 菜单顺序调整
**影响文件**: 所有 admin 页面侧边栏

- "待推送池" 菜单项移到 "调研列表" 上方
- 移除 "报告中心" 侧边栏入口
- 统一所有页面的侧边栏导航

## 📱 推荐体验流程

### 体验双视角调研列表
1. 访问 `admin/survey-list.html`
2. 点击顶部的"被评估人视角" / "评估人视角"切换按钮
3. 尝试使用不同筛选条件组合

### 体验报告预览
1. 访问 `admin/survey-list.html`
2. 点击任意已完成的调研进入详情页
3. 查看报告预览卡片，点击"查看报告"或"导出"

### 体验员工填写优化
1. 访问 `respondent/survey-form.html`
2. 查看底部重新布局的按钮
3. 填写部分内容后点击"保存"
4. 刷新页面查看草稿自动恢复
